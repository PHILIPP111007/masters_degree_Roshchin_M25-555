# project-1_Roshchin_M25-555

To install this package run:

```bash
conda env create -f ./env.yml

conda activate labyrinth_game

pip install .

# or

make install
```

To run this package use:

```bash
conda activate labyrinth_game

labyrinth_game
```

Or (if you are using poetry):

```bash
poetry run python -m labyrinth_game.main

# or

make project
```
